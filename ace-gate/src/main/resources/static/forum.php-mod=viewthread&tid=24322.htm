7 天前 上传
　　译者：玩币族Sally
　　澳大利亚珀斯铸币厂(Perth Mint)是当地最大的贵金属提炼商之一，目前正在推出由黄金支持的数字货币，以吸引消费者重新投资黄金。
　　珀斯铸币厂是澳大利亚最大的黄金提炼商，每年加工超过700吨黄金。近年来，越来越多的投资者转向使用比特币等数字货币进行新的另类投资，这刺激了这家黄金提炼商，希望通过开发自己的数字货币，使人们增加对贵金属的关注。
　　据美国广播公司新闻网报道，该公司正在开发一种尚未命名的数字货币，允许投资者在区块链上购买和出售“加密黄金”。
　　该公司首席执行官Richard Hayes在发表讲话时强调，随着投资者远离传统的避险资产，“大量资金流入比特币投资市场”。他认为，在区块链交易的由黄金支持的数字货币有其自身的优点。
　　他表示:
　　“它将具备分布式账本平台所提供的所有功能及优势，交易可以迅速完成。另外，这个数字货币是由贵金属支持的，所以实际上它不是完全意义上的‘虚拟货币’，因为有一些支持它的东西。”
　　此外，这位高管还强调，该公司有意将区块链技术应用于公司运营。通过一个应用程序追踪金条在整个供应链中的流通，包括开采到最终售出环节，所有记录都会被存储在一个透明的、不可篡改的区块链中。
　　在英国财政部宣布推出自己的“数字黄金”一年以后，该公司推出了“加密黄金”。投资者者可以通过一个名为Royal Mint gold (RMG)的数字代币，在区块链上即时购买、持有和销售黄金。区块链交易平台是由比特币初创公司BitGo联合开发，促进了威尔士加迪夫(Cardiff)附近的皇家铸币局(Royal Mint)保存的10亿美元金条的交易。
　　事实上，珀斯铸币局(Perth Mint)希望将数字货币销售平台作为他们销售黄金的另一个平台。Richard Hayes补充表示：“我们当然希望人们能够更轻松、更容易地购买贵金属。”
　　这位高管预计，这种由黄金支持的数字货币将在未来12至18个月内推出。
