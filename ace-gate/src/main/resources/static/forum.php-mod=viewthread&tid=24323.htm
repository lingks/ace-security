英国政府斥资2600万美元支持分布式账本项目

　　译者：玩币族Sally
　　1月22日，英国技术发展部门(Innovate UK)宣布， 将加倍支持区块链技术发展。
　　英国技术发展部门的相关人士表示,将投资1900万英镑(约2660万美元)用于新兴市场的新产品研发，以及支持包括分布式账本技术在内的新兴技术发展。该机构还将进一步提供1200万英镑(约1680万美元)来为那些想要招聘毕业生的企业提供资助。
　　想要得到支持的企业必须在2018年3月28日之前提交他们的项目计划，并且必须由企业或研究组织来负责运营。根据公告，这些项目需持续3个月以上，投入资本至少3.5万英镑以上，或者3年的项目投入资本需在200万英镑以上。此外，项目获得支持资金的资格标准还包括，企业应该能够自己筹集到的资金超过成本的70%。
　　这是英国技术发展部门的最新举措，该公司之前也曾在区块链领域寻求发展。去年7月，该机构向一些团体寻求帮助，旨在将区块链工具应用到新兴的卫生技术领域。
　　在此之前,英国技术发展部门向区块链初创公司提供了248,000英镑的支持，用于开发跨境金融交易工具。
　　根据公告，聚焦分布式账本技术的项目可能会与机器学习、人工智能、虚拟现实等领域的项目竞争这笔补助资金。
