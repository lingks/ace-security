日本交易所Coincheck Inc被盗，价值5亿美元的NEM不翼而飞
作者：p雨 2018-1-27 16:05 查看: 266 | 评论: 3

日本著名交易所Coincheck Inc由于在存储加密货币方式上的疏忽，沦为了黑客攻击下的牺牲品。1月26日Coincheck Inc发布消息称黑客从公司的热钱包盗走了大约价值5亿美元的数字货币（NEM）。昨晚在东京证券交易所的新闻发布会上就报道了此事件。Coincheck Inc官方人员说大约有5亿美元的NEM丢失，其他加密货币则损失很小。这件事让投资者对Coincheck Inc的信心开始动摇。

被盗消息公布后，Coincheck Inc采取了一系列的安全措施。首先，Coincheck Inc建议投资者暂时不要再冲NEM，在停止提现NEM后，随之又暂停了所有有关NEM的交易。三个半小时候，交易所又采取进一步的措施暂停了所有加密货币的提现。现在，Coincheck Inc只允许比特币进行交易，同时信用卡付款也已停止服务。

这样的事件对相信加密货币市场的投资者来说是一种打击，很多人现在还记得2013年臭名昭著的“门头沟”被盗事件。交易所被盗事件发生后市场做出的反应则是加密货币价格有所下跌。

受影响最大的当然还是NEM。根据加密货币市值网站Coinmarketcap中的数据来看，NEM全球市值排名第十。自从交易所被盗的消息传播出去之后，NEM价格就从1美元跌到0.77美元。价格的下跌，表明了负面消息的不利影响。本文写作之际，NEM价格已经恢复到0.86美元。

市场的反应一点也不令人惊讶。数字货币的丁点安全问题都会让不明所以的投资者造成恐慌。然而这样的抛售是非理性的。加密货币的定义与中心化针锋相对。所有今天发生的被盗事件强调的都是把去中心化资产存储在中心化服务中心的危险。Coincheck Inc已经承认，他们用热钱包来存储大量的NEM。Coincheck Inc如此轻率的辜负了投资者的信任。Fortune报道了该交易所的陈述：

“冷钱包对我们来说很难管理。”

对Coincheck Inc来说使用热钱包可能是便利的，但是这么做会置投资者的资金于危险的境地。这一次交易所被盗又一次警醒我们把资金放在交易所是危险的。

“如果你不掌握你的私钥，那么你就不能掌握你的加密货币。”
