Crebit征信平台：让数字资产惠及全球74亿公众
作者：admin 2018-1-31 18:52 查看: 81 | 评论: 1

  在2017年最耀眼的金融明星比特币的光芒下，区块链作为它的底层技术也已被广泛知晓并公认为是下一代如互联网般颠覆性的技术。拥有数字资产的浪潮已席卷全世界超过138个国家，即将于2018年2月启动的Crebit项目致力于真正有效利用这一产业特色，在征信领域引入人工智能信用评级，令数字资产紧贴日常生活，惠及生活中的每一步。

  区块链技术解决了两个基本的问题――时间和信任，而这正是征信领域的核心。目前征信相关的大量数据可能在传输过程中产生迟v现象，导致缺乏真实性，并非真正的有效信息。同时，作为商业命脉，少数机构垄断了大量数据，而Crebit数据的有效性则是来源于数字资产和区块链本身的全网同步，去中心化，去信任，事务数据公开透明的天然特征。它通过直接获取真实的数据―交易数据，配合人工智能的大数据分析和深度学习技术将征信标准化。Crebit将科学地计算，分析和推测每个账户的信用，推出帐户信用评分的功能，完成每个应用场景的风险预测，趋势预测，做出智能决策推荐。Crebit是一个连接虚拟世界中数字资产交易与真实世界日常生活里的信用历史记录的平台，将做到用户在全世界线上线下的信用预测，信用反欺诈。

  除了基于区块链技术的真实性和标准化，Crebit 征信平台的智能合约还具有数字资产的无疆界的特点，将可能覆盖全球74亿人。与支付宝等阿里巴巴旗下的移动金融工具优势只限于本国人口不同，Crebit是基于无国界的如比特币、以太坊等十大主流数字资产，不需要面对地域性的金融壁垒，繁琐的金融机构审核手段。根据2017年剑桥大学替代金融中心发布的《全球数字加密货币基准研究》的报告，截止2017年4月，全球已有138个国家参与数字资产交易，活跃用户已超过100万人。事实证明，数字交易的普及已在全球层面全面铺展。面对越来越多的数字资产用户，Crebit将会基于市场需求提供特有的访问功能，包括更快的查阅用户过往的交易；提供特色的金融互惠活动行销，给予广大的数字资产用户指定折扣。可预见，未来全球人口都可能受惠于Crebit信用体系。

  Crebit是为数不多的，以产品化的方式把区块链和人工智能落地到商业生活应用里面的项目。它避免了数字资产较高的涨幅波动影响，而是基于其真实的事务数据平等地对待每一位用户，建立科学标准化的征信等级，将区块链平等、民主的特点施展在征信领域。每次，无论是支付酒店押金还是贷款买房，Crebit都成为让数字资产拥有者日常生活中离不开，直接受益的金融工具。
