　　译者：玩币族Sally
　　美国证券交易委员会(SEC)主席表示，他已提示监管机构的工作人员对违反联邦证券法的ICO项目保持“高度警惕”。
　　SEC对不合规的ICO项目保持 “高度警惕”
　　美国证券交易委员会首席执行官Jay Clayton周一在美国证券监管协会(Securities Regulation Institute)的视频会议上表示，他为初创公司提供了关于如何操作合规的ICO的指导。
　　“市场专业人士，尤其是监管机构作为一个看门人的角色，需要采取负责任的行动，并坚持高标准。坦率地说，从我最近看到的情况来看，特别是在ICO领域，他们可以做得更好。”
　　Clayton解释称，“最令人不安的”的情况是，有些律师帮助初创公司将无担保证券作为ICO项目发行，这剥夺了投资者的“实质性和程序性保护要求”。
　　在ICO代币是否符合证券定义的问题上，他认为律师为初创公司给出了“模棱两可”的答案，尽管法律专业人士认为代币有可能就是一种证券。
　　Clayton警告称，他已经指示SEC工作人员对ICO项目保持“高度警惕”，尽管它们可能被包装成另外一种形式。
　　他总结道:“我已指示证交会工作人员高度警惕可能违反我国证券法精神的ICO项目。”
　　美国证券交易委员会(SEC)将对“区块链-R-Us”计划进行审查
　　不过，在谈到其他问题之前，Clayton对越来越多的上市公司表示了担忧，这些公司试图通过将自己与区块链技术联系起来，人为地抬高股价。
　　正如CCN报道的那样，生物科技公司、饮料公司甚至运动文胸生产商等公司纷纷宣布突然进军区块链市场，这导致股价飙升，而且往往是一种新的融资方式。
　　Clayton表示，该机构正在“密切关注”此类“区块链-R-Us”计划，以确定它们是否违反了证券法规。
